# BatchRefiner 简介与使用说明

## 一、BatchRefiner 是什么

BatchRefiner 是一个 AI 批量出图与改图工作台。

它适合需要批量处理图片、统一风格、反复调整、集中导出的场景。  
你不需要懂编程，只需要在网页里完成导入、配置、执行和导出。

总览界面：

![BatchRefiner 总览界面](/F:/AI%20HOME/CODEX/%E6%89%B9%E9%87%8F%E5%B7%A5%E4%BD%9C%E5%8F%B0/docs/feishu-assets/overview.png)

---

## 二、核心功能

- 批量导入图片和任务描述
- 自动生成提示词，并支持预览和手动修改
- 支持原图、全局参考图、任务参考图
- 支持单任务执行、批量执行、重新执行
- 支持平台预设与模型配置
- 支持批量导出结果图
- 支持 Edge 下写入指定下载目录

## 四、如何配置 API

打开右上角或左下角的“系统设置”，填写以下内容：

### 1. 选择平台预设

默认推荐：

- `云雾`

也支持：

- `comfly.chat`
- `通用 OpenAI 兼容`
- `Gemini / Imagen 原生`
- `自定义`

### 2. 配置接口地址

在 `API Base URL` 中填写你的中转站地址。

例如云雾默认是：

`https://yunwu.ai`

### 3. 配置 API Key

在 `API Key` 中填写你自己的密钥。

如果没有填写，网页无法正常调用模型。

### 4. 选择模型

分别设置：

- 文本模型
- 图片模型

默认推荐：

- 文本模型：`gemini-3.1-flash-lite-preview`
- 图片模型：`gemini-3.1-flash-image-preview`

### 5. 测试连接

点击“测试 API 连接”，确认接口可用后再开始正式执行任务。

## 五、如何使用

## 第一步：导入任务

支持：

- 上传图片
- 粘贴任务描述
- 拖拽图片到空白区域新建任务

如果把图片拖到任务卡片上，则会添加为该任务的参考图。

## 第二步：设置全局内容

在左侧设置：

- 全局风格
- 全局目标指令
- 全局参数
- 全局参考图

## 第三步：检查单任务差异

如果某条任务需要单独处理，可以设置：

- 任务参考图
- 单独参数

小卡片会直接显示与全局不同的内容。

## 第四步：预览提示词

点击“预览提示词”后：

- 系统生成当前任务提示词
- 提示词会直接回填
- 可继续手动修改

## 第五步：执行任务

点击“执行此项”后：

- 有提示词时直接执行
- 没有提示词时，先生成提示词再生图
- 已成功任务也可以重新执行

## 第六步：导出结果图

执行完成后可：

- 导出全部结果图
- 导出选中任务结果图
- 写入指定下载目录

---

## 六、关键规则

- 任务默认继承全局配置
- 小卡片只显示与全局不同的内容
- 拖到空白区域是新建任务
- 拖到任务卡片是添加参考图
- 已成功任务也可以再次执行

---

## 七、一句话总结

BatchRefiner 是一个把任务导入、提示词生成、参考图控制、批量执行和结果导出整合到一起的 AI 批量图片工作台。
