# BatchRefiner 更新公告

## v3.4.0

### 体验优化

- 优化多图片任务下的生成和交互流畅度。

## v2.6 - v2.8 更新归总

### 新功能

- 自定义 API 支持文本和生图分别设置地址、Key 和模型。
- 每个配置预设都支持拉取模型列表。
- 新增生成日志，方便查看任务生成、下载、导出结果。
- 结果图导出支持更完整的原图资源识别和失败提示。

### 体验优化

- API 配置区合并为更清晰的分组输入框。
- 模型候选菜单样式统一，输入区域更舒展。
- 移除画板功能，保留更直接的任务列表工作流。
- 优化批量生成的提示词缓存和结果图下载稳定性。
- 清理临时文件，仓库内容更干净。
