# BatchRefiner 更新公告

## 新功能

- 状态栏新增额度显示，可查看当前剩余额度，并通过环形进度快速判断额度余量。
- 支持导出和导入 API 配置文件，导出的配置不会直接暴露明文 Key。
- 状态栏加入版本号显示，便于确认当前网页版本。

## 体验优化

- 项目空间再次保存时会直接覆盖原文件，不再重复生成多个副本。
- 项目空间导出已去除 API 配置，分享项目更安全。
- 项目文件会附带成功任务的提示词，方便复用和归档。
- 设置页交互和提示文案已精简，界面更干净。
- 针对 Edge 做了输入框识别优化，尽量减少浏览器误判并弹出保存密码。

## 说明

- API 配置导出属于前端侧混淆保护，目的是避免直接明文暴露，不等于绝对不可提取。
- 额度环颜色规则：
  - 剩余额度占比大于等于 15% 显示绿色
  - 低于 15% 显示橙色
